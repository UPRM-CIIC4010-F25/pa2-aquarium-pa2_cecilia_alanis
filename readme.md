# PA2: Aquarium

This project is a fun multi-directionaly iteractive evolution game . This repository is part of a Programming Assignment for the Advanced Programming Course. The purpose of this project is to creatively help the student create practical programming skills by implementing changes to the base code that require strong understanding of the concepts discussed in class.

# Student Notes

Stage 2 need to knows:
-PowerUp is called "Magnet Core" and its sprite is a starfish. When activated, fishes of your same level will be attracted to you and you will serve as a magnet, thus jumping points in an instant. Lasts around 10 seconds and appears every 30 points.
-New creature fishes Nemo and Dory. When you reach level 3 Nemo appears, level 4 Dory appears and in level 5 they both appear.
-level 0 is until 10 points, level 1 is until 15 points, level 2 is until 20 points, level 3 is until 25 points, level 4 is until 35 points and level 5 is 50 points and up.
